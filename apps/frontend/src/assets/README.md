# Assets

Use this folder for source-imported assets that need IDE autocomplete and bundling.

Examples:

```tsx
import loginHero from '@assets/images/login-hero.png';
import x3Logo from '@assets/logos/x3-logo.svg';
```

Suggested folders:

- `images`: raster images used by `next/image`.
- `icons`: app-specific icons that are not from `@mui/icons-material`.
- `logos`: X3Sales logos and brand marks.
